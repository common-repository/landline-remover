== Description ==

This plugin integrates a Landline Remover API into wordpress envoirnment so users can easily access the Landline Remover API functions.
for example the user can search his/her phone number and check its Line type, DNC status, Know Litigator and remaining credits.

You can read more about this API pn thier official website [https://landlineremover.com/](https://landlineremover.com/ ) .

For terms of services visit [https://landlineremover.com/terms-of-services/](https://landlineremover.com/terms-of-services/) | For privacy policy visit [https://landlineremover.com/privary-policy/](https://landlineremover.com/privary-policy/)



== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp-landline-remover-api.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings page under Number Checker page and add you Landline Remover API key there
